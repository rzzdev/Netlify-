## ğŸš€ Cara Penggunaan (Cepat & Mudah)

Ingin menggunakan aplikasi ini? Ikuti langkah-langkah sederhana berikut:

1.  **Buka Situs Web:** Kunjungi URL situs yang sudah di-deploy.
2.  **Masukkan Nama Situs:** Ketikkan nama unik untuk situs web Anda (misal: `portofolio-saya`).
3.  **Unggah File HTML:** Pilih satu atau lebih file berekstensi `.html` atau `.htm`.
4.  **Klik Deploy!** Tekan tombol "Deploy ke Netlify" dan biarkan roket meluncur.
5.  **Selesai!** Situs Anda akan live dan Anda akan mendapatkan URL-nya.
